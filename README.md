# Prime Video Dashboard (Power BI)

## Project Overview
• This project is an interactive Prime Video Analytics Dashboard built using Power BI to analyze content data and generate meaningful insights related to movies, TV shows, genres, ratings, and release trends.  

• The goal of this project is to transform raw dataset into a visual and interactive dashboard that helps in better understanding of content distribution and platform growth.  

---

## KPIs (Key Performance Indicators)
• Total Titles: 9655  
• Total Shows: 9684
• Total Ratings: 25  
• Total Genres: 519  
• Total Directors: 5771  
• Start Date: 1920

• Imported raw dataset (CSV)  
• Data cleaning using Power Query  
• Removed duplicates  
• Handled missing/null values  
• Changed data types  
• Created calculated columns (if required)  
• Used distinct count for accurate analysis

## Visualizations Used
• Donut Chart → Movies vs TV Shows  
• Filled Map → Total Shows by Country  
• Area Chart → Total Shows by Release Year  
• Bar Chart → Ratings by Total Shows  
• Bar Chart → Genres by Total Shows  
• Cards → KPI Summary

## Features
• Interactive and dynamic visuals  
• Country-wise filtering using slicer  
• Clean dark theme dashboard (#19222D)  
• Multiple charts for deep insights

---

## Insights Gained
• Movies dominate the platform (~80%) compared to TV Shows  
• Drama and Comedy are the most popular genres  
• Significant growth in content after year 2000  
• Majority content falls under 13+ and 16+ ratings  
• Content is distributed across multiple countries  
• Rapid increase in content production in recent years
